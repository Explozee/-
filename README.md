# Currency Converter (GUI)
**Автор:** Твой Имя Фамилия

## Как запустить:
1. Установите Python.
2. Установите библиотеку requests: `pip install requests`.
3. Запустите файл: `python converter.py`.

## Особенности:
- Работает через Open ExchangeRate API (без ключа).
- Валидация ввода (только положительные числа).
- Сохранение истории в JSON.
- Удобный современный интерфейс на Tkinter.
