# Expense Tracker

**Автор:** [Кирилл Зырянов]

## Описание
Программа для контроля личных финансов. Позволяет записывать расходы, указывать категорию и дату, а также фильтровать данные и видеть общую сумму затрат.

## Как использовать
1. Запустите файл `main.py`.
2. Введите сумму расхода, выберите категорию и укажите дату.
3. Нажмите "Добавить".
4. Используйте фильтр над таблицей, чтобы увидеть расходы по конкретной категории.

## Примеры тестов
- Ввод отрицательного числа в поле "Сумма" — приложение выдаст ошибку.
- Ввод текста вместо даты — приложение попросит корректный формат (ГГГГ-ММ-ДД).
- Фильтрация по категории "Еда" — в таблице останутся только соответствующие записи, и сумма "Итого" пересчитается.
