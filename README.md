## Как получить API-ключ
1. Зарегистрируйтесь на https://exchangerate-api.com
2. Скопируйте ключ из дашборда.
3. Вставьте его в переменную `API_KEY` в начале кода.

## Тесты
- Сумма "abc" -> Ошибка "Введите положительное число".
- Сумма "-10" -> Ошибка "Введите положительное число".
- Конвертация 100 USD в RUB -> Вывод актуального курса и запись в таблицу.
