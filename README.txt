## Описание программы

Приложение для учёта личных расходов с графическим интерфейсом. Позволяет добавлять расходы по категориям и датам, фильтровать записи, подсчитывать сумму за выбранный период и сохранять данные в файл формата JSON.

## Требования

- Python 3.x.
- Библиотека `tkinter` (входит в стандартную поставку Python).
- Библиотека `datetime` (входит в стандартную поставку).
- Библиотека `json` (входит в стандартную поставку).
- Git.

## Как использовать

1. Клонируйте репозиторий.
2. Запустите файл `main.py`.
3. Введите сумму, выберите категорию и дату.
4. Нажмите «Добавить расход».
5. Используйте фильтры и подсчёт суммы на вкладке «Статистика».
6. Данные сохраняются в `expenses.json`.
